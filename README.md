<p>This addon allows you to set a suicide cost for your players it will also annouce to the other players that some one suicided <strong>"Username said goodbye cruel world"</strong>. There is 5 Convars that can changed to modifiy this addon.</p>
<h2><a id="user-content-how-to-configure" class="anchor" aria-hidden="true" href="#how-to-configure"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>How to configure</h2>
<p>By default the amount charged is random between <strong>500</strong> and <strong>1000</strong>.
You can set the minimum suicide cost by setting <strong>suicide_min_value xxx</strong> in your server console.
You can set the maximum suicide cost by setting <strong>suicide_max_value xxx</strong> in your server console.
You can disable the suicide sound it players on suicide by setting <strong>suicide_sound 0</strong> and reenable with <strong>suicide_sound 1</strong> in your server console.
You can set the suicide cost to a actual value by setting <strong>suicide_random 0</strong> and <strong>suicide_value xxx</strong> this will then charge this amount per suicide to set it back to the radom min/max set <strong>suicide_random 1</strong></p>
